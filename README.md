# Road Accident Analysis Dashboard (Power BI)
# 📊 Project Overview

This project presents an interactive Road Accident Analysis Dashboard built using Power BI. The dashboard provides insights into accident trends, casualty distribution, and key risk factors affecting road safety. It helps users quickly understand patterns related to vehicle type, road conditions, location, and time trends.

The goal of this project is to demonstrate data analysis, visualization, and storytelling skills using real-world style datasets.

# 🎯 Objectives
Analyze yearly accident and casualty trends
Identify high-risk vehicle categories
Compare Current Year vs Previous Year performance
Understand impact of road type, weather, and light conditions
Visualize accident distribution by urban and rural areas
Provide actionable insights for safety improvements
# 📌 Key Insights from Dashboard
Total Casualties: 50.5K
Total Accidents: 36.0K
Fatal Casualties: 762
Serious Casualties: 6,718
Slight Casualties: 43.1K
Majority casualties involve cars
Higher accidents observed on single carriageways
Urban areas contribute more casualties compared to rural areas
Daytime accidents are slightly higher than night accidents
# 📈 Dashboard Features
KPI cards for quick performance overview
Monthly trend comparison (Current Year vs Previous Year)
Casualties by vehicle type (Car, Bike, Bus, Van, etc.)
Road type analysis (Single, Dual, Roundabout, One way)
Urban vs Rural distribution
Light condition impact (Day vs Dark)
Interactive map for accident locations
Filters for road surface and weather conditions
